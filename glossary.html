Glossary

Character References
A character reference is a short code that, when typed into an HTML document, is read as a typeable character (&lt; = <)
Block-level Element
An element of HTML that creates its own line. If inspected in the development tool, one will find that they also create a box that follows the CSS box models. Can contain other elements, block-level or inline.
Inline Element
An element of HTML that appears within the line it is placed. Typically cant contain block elements, but can contain other inline elements.
Structural Elements
Structural elements dictate how content appears on a web page, easier allowing it to be distiguished from one another.
Semantic Elements
Sementic elements dictate the "meaning" of a web page, in the context of how its 'understood' by search engines and screen readers.
